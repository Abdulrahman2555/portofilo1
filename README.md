# Portfolio Project

A small static portfolio project containing an `index.html` and an `assets/` folder for images and other resources.

## Structure

- `index.html` — main page
- `assets/` — images, styles, and other static files

## Usage

Open `index.html` in your web browser to view the site. No build steps or server required for local viewing.

## Notes

Add more pages, update the content, or host on GitHub Pages for public access.
